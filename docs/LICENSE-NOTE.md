# License note

`LICENSE` is a project-specific permissive attribution license draft matching the requested policy:

- forks and framework redistributions are allowed
- framework redistributions must preserve AtomJS attribution and the GitHub link
- applications created with AtomJS do not have to show credit

A qualified lawyer should review the final license text before a public release. Projects that need an OSI-approved license should consider Apache-2.0 or MIT plus a separate trademark and NOTICE policy instead of a custom license.
