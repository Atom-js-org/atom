# Contributing

1. Use Node.js 20.12 or newer.
2. Run `npm install`.
3. Run `npm test` before opening a pull request.
4. Keep the public API close to Electron where the system WebView can support equivalent behavior.
5. Clearly mark intentionally unsupported behavior instead of silently pretending it works.
6. Do not add Chromium, Electron, NW.js, CEF, or Rust source to the runtime.
